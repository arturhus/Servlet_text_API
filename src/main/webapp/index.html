Description

To find word you need enter your values

Query - your word or char-combination you want to find in the file.
Limit - integer which represents max number of chars in text that API should return. If parameter is blank or missing it will 10000 chars by default.
Length - max string length that API should return string which doesn't exceed that number or if there is no such strings empty response. If parameter is blank or missing it will ignore.
